  • 1인당 최대 대출 가능 권수: 3권 → 5권

희망도서 신청 방법에 대해 안내드립니다.

신청 방법:

  1. 로그인 후 '희망도서 신청' 메뉴 접속
  2. 도서명, 저자, 출판사, ISBN 정보 입력
  3. 신청 사유 작성 후 제출

신청 가능 도서:

  • 업무 관련 전문 도서
  • 자기계발 도서
  • 일반 교양 도서

신청하신 도서는 검토 후 1-2주 내에 승인 여부를 알려드립니다.
